# JackInTheBot

## Node Version:
17+ (I use 17.8.0)

## Init setup

Clone repo in desired directory. Run `npm i` in root. 

Place a .env in the root dir with the following, you will need most of this info from a sandbox server:

DEV_TOKEN 

DEV_BOT_ID 

PROD_TOKEN=null

PROD_BOT_ID=null

IS_PROD=false


ADMIN_ROLE

MOD_ROLE

ORLANDO_GENERAL

HOLLYWOOD_GENERAL

GENERAL_CHAT

MOD_ADMIN_CHAT


SUPA_PUB_ANON
